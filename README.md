# CAN-Tranceiver
Simple breakout board for a CAN tranceiver from TI SN65HVD230D

For use with the iMX6 TinyRex baseboard lite's FlexCAN http://www.imx6rex.com/imx6-tiny-rex/baseboard-lite/
